¿Cómo conectar Notion?
1. Creá una base de datos en Notion con columnas: Título (title), Artista (text), Letra (text), Setlist (checkbox).
2. Creá una integración en notion.so/my-integrations y copiá el API Key.
3. Compartí la base de datos con tu integración.
4. Creá el proxy en Cloudflare Workers (ver INSTRUCCIONES.md incluido).
5. Pegá los datos abajo y guardá.
